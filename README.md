# Pacman Console Game (C++)

## Overview
This project is a console-based **Pacman-inspired game** developed as a **first-semester Programming Fundamentals course project**.  
It demonstrates core programming concepts through a playable terminal game built in C++.

## Project Highlights
- Real-time player movement using keyboard input
- Enemy (ghost) movement and collision detection
- Score tracking and life system
- Colored console interface for improved gameplay experience
- Win/Lose game states

## Technologies Used
- **Language:** C++14
- **Platform:** Windows Console
- **Libraries:** Standard C++ Library, `windows.h`
- **IDE:** Visual Studio Code

## Controls
- **Move Up:** `↑` or `W`
- **Move Down:** `↓` or `S`
- **Move Left:** `←` or `A`
- **Move Right:** `→` or `D`

## How to Run
1. Open the solution in **Visual Studio**.
2. Build the project in **Debug** or **Release** mode.
3. Run the executable from Visual Studio.

## Learning Objectives Covered
This project helped practice:
- Variables, conditions, loops, and functions
- Arrays/2D map handling
- Modular code organization
- Basic game loop design
- Problem solving and debugging
